Preguntas frecuentes

Sobre el desarrollo de una web.

¿Podéis renovar una web existente?

Sí, una web antigua puede ser el punto de partida. Antes de plantear el trabajo hay que revisar su estado, contenidos y objetivos actuales.

¿La web se adaptará a móviles?

El diseño responsive forma parte del planteamiento: la información y las acciones principales deben funcionar en distintos tamaños de pantalla.

¿Qué necesito preparar para pedir presupuesto?

Es suficiente con explicar tu negocio, qué quieres conseguir y qué materiales tienes. El alcance se puede ordenar después.
