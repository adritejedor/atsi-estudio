- [ ] Navegación principal.
- [ ] Rutas directas.
- [ ] Botones.

- [ ] Formularios.
- [ ] Validaciones.
- [ ] Loading.
- [ ] Éxito.
- [ ] Error.
- [ ] Estado vacío cuando aplica.

- [ ] Teléfonos y emails.

- [ ] Ortografía.

- [ ] Precios.
