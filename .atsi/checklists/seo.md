# SEO Checklist

## Por ruta

- [ ] URL clara.
- [ ] Title único.
- [ ] Description única.
- [ ] Canonical.
- [ ] Un H1.
- [ ] Headings lógicos.
- [ ] Intención de búsqueda clara.
- [ ] Copy útil.
- [ ] CTA.
- [ ] Enlaces internos.
- [ ] Open Graph.
- [ ] Imágenes con alt.

## Global

- [ ] Sitemap.
- [ ] Robots.
- [ ] 404.
- [ ] HTTPS.
- [ ] SPA fallback.
- [ ] Páginas privadas no indexadas.
- [ ] Sin enlaces rotos.
- [ ] Sin duplicación evidente.
- [ ] Datos estructurados válidos cuando aplican.
- [ ] Metadatos de marca coherentes.
- [ ] SEO local revisado cuando aplica.

## Validación

- [ ] Lighthouse SEO.
- [ ] HTML generado revisado.
- [ ] Sitemap accesible.
- [ ] Robots accesible.
- [ ] Schema validado.
