# Performance Checklist

## Medición

- [ ] Build de producción.
- [ ] Línea base registrada.
- [ ] Ruta principal medida.
- [ ] Móvil revisado.
- [ ] Consola sin errores.

## Imágenes

- [ ] Formato moderno.
- [ ] Compresión.
- [ ] Dimensiones.
- [ ] Width y height.
- [ ] Lazy loading.
- [ ] LCP no lazy.
- [ ] Sin recursos sobredimensionados.

## JavaScript

- [ ] Bundle revisado.
- [ ] Dependencias justificadas.
- [ ] Lazy routes cuando aplican.
- [ ] Sin scripts duplicados.
- [ ] Terceros revisados.

## CSS y fuentes

- [ ] Sin CSS global excesivo.
- [ ] Pocas fuentes y pesos.
- [ ] Font display.
- [ ] Animaciones eficientes.
- [ ] Sin CLS evidente.

## Red

- [ ] Caching.
- [ ] CDN.
- [ ] Compresión.
- [ ] Sin requests duplicados.
- [ ] APIs razonables.

## Resultado

- [ ] No hay regresiones.
- [ ] Mejoras medidas.
- [ ] Funcionalidad intacta.
