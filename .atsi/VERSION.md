# ATSI Workflow Version

## Current version

`1.0.0`

## Status

Primera versión funcional.

## Scope

Esta versión incluye:

- Procedimientos de análisis, implementación, revisión, optimización y despliegue.
- Estándares de Angular, SCSS, accesibilidad, SEO y rendimiento.
- Checklists de QA y release.
- Plantillas de especificación, planificación, QA y publicación.

## Versioning

Se utiliza versionado semántico:

- `MAJOR`: cambios incompatibles en el workflow.
- `MINOR`: nuevos procedimientos o capacidades compatibles.
- `PATCH`: correcciones y aclaraciones.
